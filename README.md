# Captcha-recognition-from-scratch

It's a captcha recognition Pytorch model. I used here convolution layers connected to GRU layers and it gave me 100% accuracy after 100 epochs on the validation dataset. I even didn't apply data augmentations.

Dataset looks like the following images:

![](data/efgx5.png) ![](data/g7wxw.png) ![](data/b5pnn.png) ![](data/573d8.png)

![](data/c2yn8.png) ![](data/dw3nn.png) ![](data/4b2pw.png) ![](data/bwmee.png)
